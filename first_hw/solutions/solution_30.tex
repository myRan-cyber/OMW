\paragraph{非负性}  
由定义 \( \|A\| \geq 0 \)，且 \( \|A\| = 0 \iff A = 0 \)。

\paragraph{齐次性}  
对任意标量 \( \alpha \)：
\[
\|\alpha A\| = \max_{\|x\|=1} \|\alpha Ax\|_2 = |\alpha| \max_{\|x\|=1} \|Ax\|_2 = |\alpha| \|A\|
\]

\paragraph{三角不等式}  
对任意矩阵 \( A, B \)：
\[
\|A + B\| = \max_{\|x\|=1} \|(A + B)x\|_2 \leq \max_{\|x\|=1} (\|Ax\|_2 + \|Bx\|_2) \leq \|A\| + \|B\|
\]
