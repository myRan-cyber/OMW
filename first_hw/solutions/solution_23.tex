\paragraph{加法封闭性}  
连续函数 \(f, g \in \mathcal{C}\) 的和 \(f + g\) 仍连续。

\paragraph{标量乘法封闭性}  
连续函数 \(f \in \mathcal{C}\) 的标量积 \(\alpha f\) 仍连续。

\paragraph{线性空间公理验证}  
零函数 \(0(x) = 0\) 属于 \(\mathcal{C}\)，且满足：
\begin{itemize}
    \item \(f + g = g + f\)
    \item \((\alpha \beta)f = \alpha (\beta f)\)
    \item 其他公理由实数运算性质保证。
\end{itemize}
