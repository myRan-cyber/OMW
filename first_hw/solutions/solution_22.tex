\paragraph{封闭性验证}  
对任意 \(A, B \in \mathbb{R}^{m \times n}\) 和 \(\alpha \in \mathbb{R}\)：  
1. \(A + B\) 仍为 \(m \times n\) 矩阵  
2. \(\alpha A\) 仍为 \(m \times n\) 矩阵  

\paragraph{线性空间公理}  
满足八条公理：  
- 加法交换律 \(A+B = B+A\)  
- 零矩阵存在性 \(A + 0 = A\)  
- 负元存在性 \(A + (-A) = 0\)  
- 分配律 \(\alpha(A+B) = \alpha A + \alpha B\)  
- 其余公理类似可证。
